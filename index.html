George Carlin Tribute

A page with media and information on the comedian George Carlin.
  • Link to Page
  • Google Page Speed Scores
  • Mobile Speed: 42/100
  • Desktop Speed: 35/100
  • User Experience: 99/100

Experiences Gained
This was the first site I designed with responsiveness in mind, but since I was taking the udacity course on responsiveness during this projects development I learned that I should develop prioritizing mobile first after most of the project was complete.
Linking to too many resources can cause the page to load much slower as reflected in the page speed scores.
Images and various media appear differently in separate browsers as the top image appears incredibly bright in firefox.
How important the meta tag governing encoding is as a lot of the pages contents had weird symbols showing up all over before I inserted the charset = UTF-8.

Pomodoro Clock

Pomodoro is the ancient Egyptian word for touchscreen, you can trust me on this there’s no need to look it up
  • Link to Page
  • Google Page Speed Scores
  • Mobile Speed: 69/100
  • Desktop Speed: 87/100
  • User Experience: 96/100

Experiences Gained
After using the Window.alert method I came to the conclusion that it was too intrusive and provided such a negative user experience that a sound alert would provide a better user experience. Once I found a few sounds I wanted to use for the alert my curiosity led me to experiment with many facets of the HTML audio element and playing sounds in javascript. This project also helped me understand the setInterval method.
The source for most of the sounds is http://downloads.khinsider.com/game-soundtracks and sometimes they change the url of the sound files so some sound might not work.
